# Telegram БОТ для просмотра расписания занятий

Этот бот позволяет пользователям удобно просматривать расписание занятий через Telegram. Он построен на базе библиотеки aiogram и предназначен для быстрого развертывания и использования.

## Требования
- Python 3.12 или выше
- Git
- Poetry (для управления зависимостями)

## Как развернуть проект

1. **Клонируйте репозиторий** командой:
``git clone https://github.com/frazzex/ScheduleBotAiogram.git``

2. **Перейдите в репозиторий проекта** командой: ``cd ScheduleBotAiogram``

3. **Установите Poetry** (если не установлен) командой: ``pip install poetry``

4. **Создайте виртуально окружение**, выполнив следующие действия: 
   * Выполните команду``poetry env use python``.
   
   * Затем выполните команду ``poetry env activate``.
   * В консоли отобразится путь до файла активации виртуального окружения. Вам необходимо его скопировать (без одинарных ковычек) и вставить в консоль. После этого виртуальное окружение успешно активируется.

5. **Установите зависимости**, выполнив команду ``poetry install``.

6. **Инициализируйте базу данных**, выполнив команду ``aerich init-db``.

7**Запустите бота**, выполнив команду ``poetry run python .\main.py``. Надпись **Бот запущен...** будет сигнализировать об успешном старте работы бота.

## Использование
- Добавьте бота в Telegram по ссылке: [@schedulechecker251bot](https://t.me/schedulechecker251bot).
- Отправьте команду `/start` для начала работы.
